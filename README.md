# Movie Recommender System

A content-based movie recommender system built using cosine similarity on the TMDB dataset.

## Features
- Recommends similar movies
- Uses text vectorization
- Fast similarity computation

## Tech Stack
- Python
- Pandas
- Scikit-learn

## How it works
- Data preprocessing
- Feature extraction
- Cosine similarity calculation

## How to Run
1. Run notebook to generate model files:
   - movie_list.pkl
   - similarity.pkl

2. Run app:
   streamlit run app.py
